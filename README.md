# Edfica

Edfica is a premium school operations system built with Next.js, Supabase, Tremor UI, Hugeicons, and Vercel Geist Sans.

It includes a polished dashboard, clean module pages, Tremor charts and tables, Hugeicons rendered at 2px stroke, and a Supabase schema for students, teachers, classes, attendance, fees, exams, timetables, announcements, automations, reports, and settings.

## Local setup

```bash
npm install
cp .env.example .env.local
npm run dev
```

Add your Supabase publishable key to `.env.local`.

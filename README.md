# Edfica

Edfica is a premium school operations system built with Next.js, Supabase, Tremor UI, Hugeicons, and Vercel Geist Sans.

## Included

- Clean app shell with sidebar, topbar, workspace search, alerts, and quick actions
- Tremor dashboard with KPI cards, charts, tracker, tables, badges, progress UI, inputs, and date controls
- Hugeicons rendered through a reusable AppIcon wrapper with 2px stroke
- Geist Sans font setup
- Supabase client scaffold and school operations schema
- Modules for students, teachers, classes, attendance, fees, exams, timetable, announcements, automations, reports, and settings

## Local setup

```bash
npm install
cp .env.example .env.local
npm run dev
```

Add the Supabase project URL and publishable client key to `.env.local` before connecting live data.

## Design rules

Tremor for UI, Hugeicons for icons, Geist Sans for font, clean white surfaces, subtle borders, calm status colors, and rules-based automations without AI.
